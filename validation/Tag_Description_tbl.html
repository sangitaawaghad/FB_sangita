Tsg Name Tag Description
"<"input type='text'"/>" 
-Specifies the input field where user can enter the data 
-User can enter the text input
"<"input type='password'"/>" 
-Specifies the input Field where user can enter the data
-User can enter the password which display in dot format
"<"input type='button'"/>" 
-used to handle javascript event at client side.
-it is used to for simple calculation purpose
"<"input type='submit'"/>" 
-defines button for submitting data to form handler
- it is used within form tag only





	"<"input type='reset'"/>"
	
-it is used to reset form controls.it clears all textbox on form.and it is used with only form tag






	"<"form action='second.html' method='get/post' enctype=''"/>"
	
-used for creating form for user input
-action attribute specifies the url of back end script that will process the form.
-method attribute specifies the HTTP method to use when form is submitted.in get method form data is appended to 
url when form data is submitted.in post method form data is not appended to url when form is submitted
-Enctype attribute specifies the mine type used to encode content of the form.by defualt is url encoded. 
url encoded does not support binary data
